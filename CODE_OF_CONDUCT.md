# Código de Conduta

## Nosso Compromisso

Este é um espaço de aprendizado intenso e respeito mútuo.

## Padrões

- **Respeito** acima de tudo
- **Críticas** são bem-vindas se forem sobre o código, não sobre a pessoa
- **Dúvidas** são incentivadas — ninguém nasce sabendo ponteiros
- **Zero tolerância** a assédio, discriminação ou desrespeito

## Escopo

Aplica-se a issues, discussões e qualquer interação neste repositório.
