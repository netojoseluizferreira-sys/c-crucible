# Contribuindo para The C Crucible

## 🚫 Este repositório NÃO aceita contribuições externas

The C Crucible é um projeto de forja pessoal. Cada linha de código aqui
representa meu próprio suor, meus próprios erros e minhas próprias vitórias.

## 📖 No entanto, você pode:

- **Estudar** o código à vontade
- **Abrir issues** para discutir soluções alternativas
- **Forkar** o repositório e trilhar sua própria jornada
- **Compartilhar** com outros estudantes que queiram seguir o caminho do C

## ⚠️ Aviso

Todo o código aqui é compilado com `-Wall -Wextra -Werror -pedantic`.
Se for usar como referência, mantenha o mesmo padrão de qualidade.
